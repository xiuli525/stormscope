# StormScope 增强功能设计方案

## 概述

为 StormScope 新增 3 个精品功能，提升数据深度、实用性和社交分享能力。

---

## 功能一：天文与生活面板

### 位置

Dashboard 页面新增卡片区域，或新建独立页面 `/lifestyle`。

### 子模块

**日出日落可视化**

- SVG 太阳弧线动画，显示日出到日落的完整弧线
- 实时标注当前太阳位置（按当前时间插值计算）
- 显示日出、日落、白昼时长
- 数据来源：Open-Meteo daily API 的 `sunrise` / `sunset` 字段（已有）

**月相显示**

- 根据日期计算月相（新月→满月→新月周期 29.53 天）
- 对应月相图标 + 名称（新月、蛾眉月、上弦月、盈凸月、满月、亏凸月、下弦月、残月）
- 纯前端算法，无需 API

**穿衣建议**

- 综合温度 + 风速 + 湿度 + 紫外线指数
- 输出分级建议：短袖/长袖/薄外套/厚外套/羽绒服 + 防晒/雨具提醒
- 配合图标展示

**运动建议**

- 综合 AQI + 温度 + 降水概率 + 风速
- 输出适宜/一般/不适宜 + 具体建议（如"适合跑步"、"建议室内运动"）

### 技术方案

- 数据：复用已有 weather/air-quality hooks
- 日出日落弧线：SVG path + CSS animation
- 月相算法：纯前端数学计算
- 新依赖：无

## 功能二：极端天气预警系统

### 检测规则

| 类型 | 阈值                | 等级 |
| ---- | ------------------- | ---- |
| 暴雨 | 日降水 > 50mm       | 红色 |
| 大雨 | 日降水 > 25mm       | 橙色 |
| 高温 | 日最高 > 35°C       | 橙色 |
| 酷热 | 日最高 > 40°C       | 红色 |
| 寒潮 | 相邻两天降温 > 10°C | 橙色 |
| 大风 | 风速 > 60km/h       | 橙色 |
| 暴风 | 风速 > 90km/h       | 红色 |

### UI 设计

- 页面顶部滑入式横幅（Framer Motion 动画）
- 颜色分级：黄色（关注）、橙色（警告）、红色（严重）
- 图标 + 简短描述 + 影响时段
- 可点击展开详情
- 无预警时不显示

### 技术方案

- 数据：复用 7 天预报数据（useForecast hook）
- 新增 `utils/weather-alerts.ts` 检测逻辑
- 新增 `components/weather/WeatherAlerts.tsx` UI 组件
- 新依赖：无

## 功能三：天气分享卡片生成器

### 功能

- 将当前天气数据渲染为精美卡片图片
- 一键下载为 PNG
- 包含：城市名、温度、天气图标、天气描述、日期、背景渐变

### 模板风格（3 种）

1. **极简白** — 白底 + 大字温度 + 简洁线条
2. **渐变卡** — 天气对应的渐变背景 + 毛玻璃信息层
3. **像素风** — 8-bit 风格像素天气图标 + 复古配色

### 技术方案

- 使用 html2canvas 将 DOM 节点转为 Canvas 再导出 PNG
- 卡片尺寸：1080×1920（手机壁纸比例）和 1200×630（社交分享）
- 带 "StormScope" 小水印
- 新增 `components/weather/ShareCard.tsx`
- 新增 `pages/Share.tsx` 或 Dashboard 内弹窗
- 新依赖：html2canvas
